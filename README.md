## Highlighters 😎
<!-- 대충 이미지 -->

 <h3 align="left">Highlighters는 3가지 고민에서 시작되었습니다.</h3>

- 링크를 브라우저에서 곧바로 팀원에게 보낼 수 없을까? 
- 공유한 링크를 다시 찾기 쉽게 모아두면 좋겠어! 
- 글에서 중요한 내용이 한눈에 보였으면 좋겠어!
